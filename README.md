## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/spend-billgates-money-clone.git
   cd spend-billgates-money-clone
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## How to Play

1. You start with Bill Gates' net worth as your balance
2. Browse through various items you can purchase
3. Use the quantity selector to choose how many items you want to buy/sell
4. Watch your balance update in real-time as you spend
5. Try to spend it all!
